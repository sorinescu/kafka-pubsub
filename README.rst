Install
-------

Directly from PyPI:

.. code:: bash

    $ pip install kafka-pubsub

Or locally, in development mode:

.. code:: bash

    $ git clone https://github.com/sorinescu/kafka-pubsub.git
    $ cd kafka-pubsub
    $ pip install -e .
